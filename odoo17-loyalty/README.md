# Customer Loyalty Points for Odoo 17

Este repositorio contiene un módulo profesional para Odoo 17 que implementa un sistema de puntos de fidelidad para clientes.

## 🎯 Objetivo
Desarrollado para gestionar reglas de fidelización, transacciones de puntos e integración con el flujo de ventas.

## 🚀 Características
- **Reglas de Puntos**: Define ratios de obtención y redención por monto.
- **Transacciones**: Historial completo de puntos ganados y gastados.
- **Ventas**: Acreditación automática al confirmar pedidos y descuentos por puntos.
- **Reportes**: Resumen de puntos en PDF.
- **Docker Ready**: Configuración lista para desplegar.

## 🛠️ Tecnologías
- **Odoo 17.0** (Community/Enterprise)
- **Python 3.10+**
- **PostgreSQL 16**
- **Docker & Docker Compose**

## 📦 Instalación
1. Clonar el repositorio:
   ```bash
   git clone https://github.com/lilyreyesvargas-max/odoo17-loyalty.git
   ```
2. Levantar el entorno:
   ```bash
   docker-compose up -d
   ```
3. Acceder a `http://localhost:8069`.
