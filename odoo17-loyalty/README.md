# Customer Loyalty Points for Odoo 17

This repository contains a professional module for Odoo 17 that implements a loyalty points system for customers.

## 🎯 Objective
Developed to manage loyalty rules, points transactions, and integration with sales workflow.

## 🚀 Features
- **Loyalty Rules**: Define earning and redemption ratios by amount.
- **Transactions**: Full history of earned and spent points.
- **Sales**: Automatic accrual when confirming orders and discounts by points.
- **Reports**: Summary of points in PDF.
- **Docker Ready**: Ready to deploy configuration.

## 🛠️ Technologies
- **Odoo 17.0**  (Community/Enterprise)
- **Python 3.10+**
- **PostgreSQL 16**
- **Docker  & Docker Compose**

## 📦 Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/lilyreyesvargas-max/odoo17-loyalty.git
   ```
2. Start the environment:
   ```bash
   docker-compose up -d
   ```
3. Access `http://localhost:8069`.
