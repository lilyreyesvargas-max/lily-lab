# Customer Loyalty Points

Módulo avanzado para gestionar la fidelización de clientes mediante un sistema de puntos.

## Funcionalidades Principales

### 1. Reglas de Acumulación
Configura cuántos puntos gana un cliente por cada unidad monetaria gastada. Puedes definir montos mínimos de compra para activar la acumulación.

### 2. Redención de Puntos
Permite a los clientes canjear sus puntos por descuentos directos en sus pedidos de venta. El ratio de conversión es configurable por el administrador.

### 3. Historial de Transacciones
Cada movimiento de puntos (ganancia o gasto) queda registrado con su origen (ej. número de pedido) y fecha, garantizando total trazabilidad.

### 4. Reporte de Fidelidad
Genera reportes en PDF con el estado actual de los puntos del cliente y su historial de movimientos.

## Configuración de Seguridad
- **Loyalty Manager**: Acceso total para configurar reglas y ver todas las transacciones.
- **Usuario Interno**: Puede ver el saldo de puntos del cliente.

## Instalación
Este módulo depende de:
- `base`
- `contacts`
- `sale_management`
